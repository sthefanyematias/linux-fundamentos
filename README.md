# Linux – Fundamentos

Este repositório reúne estudos práticos em Linux, com foco no uso real da linha de comando,
organização do sistema, automação básica e análise operacional.

## Conteúdo

- Fundamentos do sistema Linux
- Manipulação de arquivos e diretórios
- Permissões, usuários e grupos
- Gerenciamento de processos e sinais
- Scripts Bash para automação de tarefas
- Comandos de rede e inspeção de conectividade
- Leitura e análise básica de logs

> Ambiente de estudo: **Debian GNU/Linux 13.2.0**

Cada diretório contém um README próprio descrevendo o escopo dos exercícios
e o objetivo prático de cada conjunto de comandos ou scripts.
