# Scripts Bash

Este diretório contém scripts simples desenvolvidos para praticar automação
de tarefas comuns no Linux.

Os scripts foram escritos com foco em clareza e entendimento do funcionamento,
não em complexidade excessiva.

#

### Como Utilizar os Scripts
Para executar qualquer script deste diretório, siga o passo a passo abaixo. Utilizaremos o `01.backup.sh` como exemplo:

1. Preparação do Arquivo
   - Certifique-se de que o código está salvo no arquivo correspondente. Se estiver criando um novo, você pode usar o editor `nano`:

```bash

nano backup.sh

```

2. Atribuição de Permissão
   - Por padrão, arquivos de texto não são executáveis no Linux. Você precisa conceder essa permissão ao seu usuário:

```bash

chmod +x backup.sh

```

3. Execução
   - Agora, basta chamar o interpretador Bash para rodar o arquivo:

```bash

./backup.sh

```
