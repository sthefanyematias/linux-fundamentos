# Gerenciamento de Arquivos e Diretórios

### Criar arquivos e diretórios

1. `touch`
   - cria um arquivo vazio ou atualiza a data de acesso de um existente.

```bash

touch arquivo.txt

```

2. `mkdir` (Make Directory)
   - cria uma nova pasta.

```bash

mkdir diretorio

```

#

### Copiar e mover (ou Renomear)

1. `cp` (Copy)
   - copia arquivos ou diretórios.

```bash

cp arquivo.txt destino/

```

2. `mv` (Move)
   - move arquivos ou altera o nome.

```bash

mv arquivo.txt novo_nome.txt

```

#

### Remover

1. `rm` (Remove)
   - apaga arquivos.

```bash

rm arquivo.txt

```

- `rm -r`: apaga diretórios e todo o seu conteúdo (recursivo).
- `rm -i`: modo interativo (pede confirmação antes de apagar).
- nunca execute `rm -rf / —` ou este comando tentará apagar todo o seu sistema de arquivos.
