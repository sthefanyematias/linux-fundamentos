# Comandos Básicos

### Navegação no sistema de arquivos

```bash

pwd

```

- `pwd`: exibe o diretório atual

```bash

ls

```

- `ls`: lista arquivos e diretórios

```bash

ls -la

```

- `ls -la`: lista detalhada, incluindo arquivos ocultos

```bash

cd /caminho/diretorio

```

- `cd`: altera o diretório atual

### Ajuda e documentação

```bash

man ls

```

- `man`: exibe a página de manual completa do comando

```bash

ls --help

```
- `--help`: mostra uma ajuda resumida diretamente no terminal
