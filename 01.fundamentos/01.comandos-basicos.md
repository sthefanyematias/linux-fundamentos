# Comandos Básicos

### Navegação no sistema de arquivos


1. `pwd` (Print Working Directory):
   - exibe o caminho completo do diretório onde você está no momento.

```bash

pwd

```

 2. `ls` (List):
    - lista os arquivos e pastas do diretório.

```bash

ls

```

- `ls`: listagem simples.
- `ls -l`: listagem detalhada (mostra permissões, tamanho e data).
- `ls -la`: mostra arquivos ocultos (que começam com .).
- `ls -lah`: detalhado e com tamanhos em formato human readable (Ex: 2K, 10M, 1G).

3. `cd` (Change Directory):
   - utilizado para navegar entre as pastas.

```bash

cd /caminho/diretorio

```

- `cd ..`: volta um nível para a pasta pai.
- `cd ~`: volta direto para a pasta pessoal (Home) do seu usuário.
- `cd -`: volta para o diretório onde você estava antes do último comando.

#

### Ajuda e documentação

1. `man` (Manual)
   - exibe a página de manual completa do comando

```bash

man ls

```

2. `--help` (Help)
   - mostra uma ajuda resumida diretamente no terminal
     
```bash

ls --help

```

3. `whatis` (What is)
   - exibe uma definição curta e direta sobre o que o comando faz.
     
```bash

whatis mkdir

```
